U.S. Bee Colony
Health Explorer

A fully interactive dashboard built on USDA colony health data spanning 2015–2023. Select any metric, compare states by clicking them on the map, and trace trends over time in the chart alongside it.

Stack D3.js · Python ETL · GitHub Pages
Data 50-state coverage, quarterly records
Pipeline Automated via GitHub Actions on every push

How to use the dashboard

  1. 1
    Pick a metric from the dropdown above the visualizations. The map color gradient updates to show national averages for that metric.
  2. 2
    Click a state on the map to select it — a highlighted border marks it and its time-series line appears in the chart on the right.
  3. 3
    Add more states by clicking additional ones. The chart overlays all selected states for a direct side-by-side comparison.
  4. 4
    Hover the chart to reveal exact values per state at any point in time via the crosshair overlay.
  5. 5
    Clear Selection resets the chart back to the national view.
